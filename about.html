Bio

My name is Travis Amador. I am from a small town in Michigan called Brown City. It is a town most people have never heard of, mostly because it's so small, and there's not much to it aside from farms and fields. Before I started learning coding, I worked in landscaping. In that job, I mostly ran small lawn crews everywhere east of Lansing from the tip of the thumb to the Ohio border in Michigan, but working in and around Pontiac most often.




My Journey Into Coding

After joining the Grand Circus c#/.Net full stack bootcamp, I have started learning code at a pace I previously could not imagine. So far I have learned C#, HTML, SQL, .NET Core, ASP.NET core and ASP.NET Core MVC, Entity Framework, implementing and creating APIs with ASP.NET, Bootstrap, razor, Git and Github. In C# I have learned good coding practices, how to use linq, object oriented programming and the use of OOP MVC in ASP.NET. In SQL I have learned how to work with databases using queries, as well as how to use databases in C# With Entity framework. With HTML I have learned how to build websites both in just HTML and using HTML in conjunction with C# with razor in ASP.NET.
